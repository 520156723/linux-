# curl
- -o 表示将返回输出到文件中
  - 示例 $ curl -o linux.html http://www.linux.com 将结果写到linux.html中
- -m 设置最大传输时间
- -s 不输出任何东西
- -w 获取请求结果，常用于诊断
  - 示例 curl -s -w %{http_code} 获取http响应码
